# Proxy Configs

Personal proxy configuration assets for Stash and Loon.

## Structure

- `configs/loon/` - complete Loon configuration files.
- `rules/stash/` - Clash/Stash YAML rule providers.
- `rules/loon/` - Loon `.lsr` rule providers.
- `assets/icons/` - icons used by proxy groups.

## Rule Files

- `Custom_Direct` - custom direct rules.
- `Apple_Push` - APNs push notification rules based on Stash FAQ and Apple's APNs network ranges.
